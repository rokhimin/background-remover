### ReadMe
# Background Remover

**Background Remover** is a web application designed to remove the background from images, providing a clean and transparent result. This tool is particularly useful for graphic designers, photographers, and anyone needing to isolate subjects from their backgrounds.

## Library
- Jekyll
- Bulma css & Font-Awesome
- Javascript Canvas

## Features

- **Automatic Background Removal**: Utilizes advanced algorithms to detect and remove backgrounds without manual intervention.
- **User-Friendly Interface**: Simple and intuitive design for easy navigation and operation.
- **Fast Processing**: Quickly processes images to deliver results in seconds.

## Development

To set up a development environment for this project, follow these steps:

1. **Clone the Repository**:
   ```bash
   git clone https://github.com/rokhimin/background-remover.git
   ```
2. **Navigate to the Project Directory**:
   ```bash
   cd background-remover
   ```
3. **Install Dependencies**:
   Ensure you have [Bundler](https://bundler.io) installed, then run:
   ```bash
   bundle install
   ```
4. **Serve the Application Locally**:
   Start a local server to view the application:
   ```bash
   bundle exec jekyll serve
   ```
   Open your browser and navigate to `http://localhost:4000` to see the application in action.
